Search Demo

This page contains several demos of the search component of the new BYU Header. Note that many of these are incomplete pending further development.

Send search to URL

<byu-search
  action='https://www.google.com/#q=$1'
  method='get'
  placeholder='Search Google'>
</byu-search>

Send search to URL and provide input element

<byu-search action='https://www.google.com/#q=$1'>
  <input slot='search' type='search' placeholder='Search Google'>
</byu-search>

Set Search Target

<byu-search
  action='https://www.google.com/#q=$1'
  target='_blank'
  placeholder='Search Google'>
</byu-search>

Set Initial Value

<byu-search value='foo'></byu-search>

<byu-search>
  <input slot='search' type='text' placeholder='My Input' value='foo'>
</byu-search>

Listen for Input Events

<script>
function changeListener(element) {
  console.log(element, element.value);
}
</script>

<byu-search oninput='changeListener(this)'></byu-search>

<byu-search>
  <input slot='search' type='search' oninput='changeListener(this)'>
</byu-search>

Listen for Search Events

<byu-search onsearch='changeListener(this)'></byu-search>

<byu-search onsearch='changeListener(this)'>
    <input type='search'>
  </byu-search>
